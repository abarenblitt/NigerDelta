This folder contains raster files produced in paper Fatoyinbo et al 2026

Files:
AmonLoss (7 files): Unclassified loss of vegetation detected from anomolous change in NDVI during the epoch listed in file name
ClassificationOutput (6 files): Anomolous loss classified by loss driver, results of Random Forest Classification described in paper (1 = Oil, 2 = Erosion, 3 = Urban)
Classified_ALOS (2 files): Mangrove region classified as Nypa (1), Water (4), or Mangrove (5)
NigeriaMangrove2019: Mangrove extent in 2019, results of Random Forest Classification described in paper
